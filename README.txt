# Keepit Chrome Extension

Welcome to Keepit, a user-friendly Chrome extension designed to enhance your note-taking experience right within your browser.

## Installation

To install Keepit from this GitHub repository onto someone's browser, follow these steps:

1. **Clone the Repository:** Begin by cloning this GitHub repository to your local machine.

2. **Load the Extension in Chrome:**
- Open Google Chrome and type `chrome://extensions/` in the address bar.
- Enable "Developer mode" by toggling the switch in the top-right corner of the page.
- Click the "Load unpacked" button and select the folder where you cloned the repository (`keepit-extension`).

3. **Verify Installation:**
- The Keepit extension icon should now appear in your Chrome toolbar.
- Click the icon to open the Keepit interface and start adding notes.

## Usage

With the Keepit extension, you can effortlessly store and manage your important notes directly within your browser. Enjoy the convenience of accessing and organizing your notes while browsing. Happy note-taking with Keepit!
